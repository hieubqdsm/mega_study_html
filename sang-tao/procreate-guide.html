Phần 1 · Giao diện & Cử chỉ — Làm quen từ đầu
Interface Canvas — Vùng vẽ Bước 0 ▼

Vùng trắng chính là nơi bạn vẽ. Khi tạo canvas mới, chọn kích thước phù hợp với mục đích: vẽ minh họa in ấn cần độ phân giải cao (300 DPI), vẽ cho màn hình dùng 72–150 DPI.

// Tạo canvas mới: Góc phải trên → dấu + Kích thước khuyên dùng: A4 300dpi → In ấn, minh họa 2048×2048 px → Avatar, icon 1920×1080 px → Wallpaper, poster Screen Size → Phác thảo nhanh
⚠️ Canvas càng lớn + DPI cao → càng ít layer tối đa. Kiểm tra số layer cho phép trước khi vẽ dự án lớn.
Interface Toolbar — 4 nút góc màn hình Bước 0 ▼

Procreate có giao diện cực kỳ tối giản. 4 icon trên cùng bên phải điều khiển mọi thứ.

🔧 Actions → Cài đặt canvas, import, export, share ✦ Adjustments → Bộ lọc màu, blur, hiệu ứng ⟲ Selection → Chọn vùng để di chuyển / xóa ⤢ Transform → Scale, xoay, warp đối tượng
💡 Sidebar trái: thanh trượt trên là Brush Size, dưới là Opacity. Nút tròn giữa là Undo/Redo.
Gesture Gesture cơ bản — Shortcut ngón tay Bước 1 ▼

Procreate được tối ưu cho cử chỉ ngón tay. Học thuộc các gesture này để vẽ nhanh gấp đôi.

// Cơ bản 2 ngón chạm → Undo (hoàn tác) 3 ngón chạm → Redo (làm lại) Pinch in/out → Zoom canvas 2 ngón xoay → Xoay canvas // Nâng cao 3 ngón vuốt xuống → Copy / Paste menu Giữ & chạm 2 ngón → Undo liên tục Vuốt tay lên canvas → Eyedropper (hút màu)
🎯 Giữ ngón tay khi vẽ đường thẳng → QuickShape tự động chỉnh thành hình học chuẩn.
Gesture QuickShape — Vẽ hình chuẩn tự động Bước 1 ▼

Vẽ bất kỳ hình dạng nào rồi giữ nguyên bút — Procreate sẽ tự làm đẹp hình đó thành vòng tròn, hình chữ nhật, tam giác, đường thẳng hoàn hảo.

// Cách dùng Vẽ hình → Giữ bút không nhấc → QuickShape kích hoạt Sau đó: Chạm thêm 1 ngón tay → Edit Shape (chỉnh chính xác) Thả ngón tay → Giữ nguyên hình đã làm đẹp
✅ Kết hợp QuickShape + Edit Shape để vẽ hình học chính xác mà không cần tool riêng.
Phần 2 · Brush — Công cụ vẽ
Brush Brush Library — Thư viện cọ Bước 0 ▼

Procreate có hàng trăm cọ mặc định chia theo nhóm. Bấm icon cọ trên góc phải để mở.

// Nhóm brush mặc định hay dùng Sketching → 6B Pencil, HB Pencil (phác thảo) Inking → Studio Pen, Syrup (tô màu phẳng) Painting → Acrylic, Gouache (vẽ tranh) Airbrushing→ Soft Airbrush (tô bóng mượt) Calligraphy→ Monoline (chữ đều nét)
📌 Import brush .brushset miễn phí từ internet: Actions → Add → Import
Brush Brush Size & Opacity — Kích thước & Độ trong Bước 1 ▼

Hai thanh trượt bên trái canvas điều khiển size và opacity của cọ đang dùng.

// Sidebar trái Thanh trên → Brush Size (1% → 100%) Thanh dưới → Opacity (1% → 100%) // Apple Pencil Lực nhấn mạnh → Nét to / đậm hơn Nghiêng bút → Hiệu ứng shading (với brush hỗ trợ)
💡 Nhấn đúp vào nút tròn undo/redo ở giữa sidebar để reset opacity về 100%.
Brush Studio Brush Studio — Tùy chỉnh cọ Bước 2 ▼

Bấm vào tên brush để mở Brush Studio — nơi chỉnh chi tiết mọi thứ của cọ. Đây là nơi tạo brush riêng.

// Các tab quan trọng trong Brush Studio Stroke Path → Khoảng cách, độ mượt nét vẽ Taper → Đầu nét nhọn / tù (calligraphy) Shape → Hình dạng đầu cọ (tròn / vuông / custom) Grain → Texture bề mặt cọ Dynamics → Jitter, speed, wobble Apple Pencil → Lực nhấn → Size / Opacity / Bleed
✅ Duplicate brush trước khi chỉnh để không mất bản gốc: vuốt trái tên brush → Duplicate.
Brush Studio Tạo Brush từ ảnh — Custom Stamp Brush Bước 3 ▼

Bạn có thể biến bất kỳ ảnh nào thành brush: hoa, lá, kết cấu vải, mây... Rất hữu ích cho vẽ background nhanh.

// Quy trình tạo stamp brush 1. Chuẩn bị ảnh PNG nền trắng (hoặc trong suốt) 2. Brush Library → + góc phải trên 3. Brush Studio → Shape → Edit → Import 4. Vào Grain → chọn Source (nếu cần texture) 5. Chỉnh Spacing, Scatter để ra hiệu ứng rải đẹp 6. Đặt tên → Done
🎨 Brush stamp lá cây + Scatter cao + Random hue → tạo cành lá cực nhanh.
Phần 3 · Layer — Lớp vẽ
Layer Layer Panel — Bảng quản lý lớp Bước 0 ▼

Layer là trụ cột của Procreate. Mỗi layer là một "tờ giấy trong suốt" xếp chồng. Bấm icon layer (2 hình vuông) để mở.

// Thao tác layer cơ bản Bấm + → Thêm layer mới (phía trên layer đang chọn) Vuốt trái → Lock / Duplicate / Delete Kéo thả → Sắp xếp lại thứ tự Bấm tên layer → Rename, chọn Blend Mode, thêm Mask Tích checkbox → Ẩn / hiện layer
⚠️ Số layer tối đa phụ thuộc RAM + kích thước canvas. iPad Pro có nhiều layer hơn iPad mini.
Layer Alpha Lock — Tô màu trong vùng đã vẽ Bước 2 ▼

Alpha Lock khóa vùng trong suốt của layer — chỉ có thể vẽ thêm lên phần đã có nét/màu. Cực kỳ hữu ích để tô bóng mà không ra ngoài.

// Bật Alpha Lock Cách 1: Bấm tên layer → Alpha Lock Cách 2: Vuốt phải 2 ngón trên layer (nhanh hơn!) // Khi Alpha Lock bật → Background layer thumbnail có dạng kẻ ô bàn cờ → Mọi nét vẽ chỉ nằm trong vùng pixel có màu
💡 Dùng Alpha Lock + Airbrush để tô bóng nhân vật nhanh: không cần selection, không tràn ra ngoài.
Layer Group Layer — Nhóm các layer Bước 2 ▼

Khi số layer nhiều, gom lại thành group để dễ quản lý. Nhân vật một nhóm, background một nhóm.

// Tạo group Cách 1: Chọn nhiều layer (vuốt phải từng layer) → Group Cách 2: Kéo layer này đè lên layer kia → tự tạo group // Lợi ích → Move cả group cùng lúc → Transform toàn bộ nhân vật như 1 khối → Blend Mode cho cả group → Flatten group khi xong → tiết kiệm layer slot
✅ Đặt tên group rõ ràng: "Nhân vật", "Background", "FX", "Text".
Blend Mode Blend Mode — Chế độ hoà trộn layer Bước 3 ▼

Blend Mode quyết định cách layer hoà vào layer bên dưới. Đây là bí quyết tạo ánh sáng, bóng và hiệu ứng màu đẹp.

// Các Blend Mode quan trọng nhất Multiply → Tối màu (tô bóng, shadow) Screen → Sáng màu (hiệu ứng ánh sáng) Overlay → Tăng tương phản, sắc màu Add → Glow, ánh sáng neon cực sáng Soft Light → Bóng mềm, tinh tế Color → Đổi màu giữ nguyên độ sáng Luminosity → Đổi độ sáng giữ nguyên màu
🎨 Quy trình tô bóng chuyên nghiệp: Layer Multiply 40–60% opacity → tô bóng xanh/tím. Layer Add 30% → chấm highlight trắng.
Phần 4 · Color — Màu sắc
Color Color Panel — 5 chế độ chọn màu Bước 0 ▼

Bấm vòng tròn màu góc phải trên để mở Color Panel. Có 5 tab chọn màu khác nhau.

Disc → Bánh xe màu trực quan, dễ dùng nhất Classic → Ô vuông HSB — kiểm soát chính xác Harmony → Gợi ý bảng màu hài hoà tự động Value → Nhập HEX / RGB / HSB chính xác Palettes → Lưu bảng màu dự án
📌 Tạo Palette từ ảnh: Palettes → + → New from Photos → chọn ảnh → Procreate tự trích 5 màu chủ đạo.
Color Fill ColorDrop — Đổ màu vào vùng kín Bước 1 ▼

Kéo vòng tròn màu thả vào vùng kín bất kỳ trên canvas → tô màu tức thì. Đây là kỹ thuật flat color nhanh nhất.

// ColorDrop nâng cao Kéo thả + giữ → Hiện thanh "threshold" → Kéo phải: threshold cao → fill tràn ra nhiều hơn → Kéo trái: threshold thấp → fill bó chặt đường kẻ Reference Layer → Fill sang layer khác dựa theo lineart → Bật tính năng: Layer lineart → bấm tên → Reference → Tạo layer mới bên dưới → fill sẽ "nhìn" vào Reference
💡 Reference Layer là workflow tô màu chuyên nghiệp: lineart 1 layer, màu layer riêng, không bao giờ lẫn vào nhau.
Color Eyedropper — Hút màu từ canvas Bước 1 ▼

Lấy màu từ bất kỳ điểm nào trên canvas để dùng tiếp.

// Kích hoạt Eyedropper Cách 1: Giữ ngón tay lên canvas (không dùng bút) Cách 2: Bấm nút vuông nhỏ trên sidebar (nếu bật) Cách 3: Actions → Prefs → Gesture Controls → Eyedropper // Vòng tròn hiện ra → Nửa trên: màu đang hút → Nửa dưới: màu hiện tại đang dùng
🎯 Hút màu từ ảnh tham chiếu import vào canvas → dựng bảng màu chính xác cho dự án.
Phần 5 · Transform · Selection · Drawing Guides
Transform Transform Tool — Di chuyển & biến đổi Bước 1 ▼

Bấm icon mũi tên (⤢) để vào Transform. Chọn nội dung layer và biến đổi theo nhiều cách.

// 4 chế độ Transform Freeform → Kéo tự do (méo được) Uniform → Scale giữ tỉ lệ (khuyên dùng nhất) Distort → Kéo 4 góc độc lập (phối cảnh) Warp → Lưới biến dạng tự do (uốn cong) // Tips Bật Snapping → canh giữa / canh đều tự động Flip Horizontal → lật nhân vật qua phải
⚠️ Transform sẽ bị vỡ ảnh nếu scale lên lớn hơn kích thước gốc. Vẽ to trước, thu nhỏ sau.
Selection Selection Tool — Chọn vùng Bước 2 ▼

Bấm icon (⟲) để chọn vùng. Có 4 loại selection tool.

Automatic → Chọn vùng màu giống nhau (như magic wand) Freehand → Vẽ tay vùng chọn tùy ý Rectangle → Chọn hình chữ nhật Ellipse → Chọn hình oval // Sau khi có selection → Transform: di chuyển vùng chọn → Copy & Paste: 3 ngón vuốt xuống → Copy, Paste → Fill: ColorDrop vào vùng chọn → Clear: vuốt 3 ngón → Clear
✅ Freehand selection + Feather cao → xóa / cắt dần mờ, không cứng cạnh.
Drawing Guides Drawing Guides — Lưới hỗ trợ vẽ Bước 2 ▼

Actions → Canvas → Drawing Guide → Edit Drawing Guide. Công cụ cực mạnh giúp vẽ đúng tỉ lệ, phối cảnh, đối xứng.

2D Grid → Lưới vuông thông thường Isometric → Lưới isometric (vẽ game 2.5D) Perspective → 1/2/3-point perspective tự động Symmetry → Đối xứng (vẽ 1 bên tự có bên kia) // Assisted Drawing → Bật "Assisted" trên layer → nét vẽ tự snap theo guide → Vẽ đường thẳng hoàn hảo không cần ruler
🎨 Symmetry + Radial = vẽ mandala tự động. Vẽ 1 phần → có ngay đối xứng 8/16 chiều.
Phần 6 · Mask & Adjustments — Hiệu ứng nâng cao
Mask Layer Mask — Che/hiện layer không phá hủy Bước 3 ▼

Mask là layer đen/trắng đính kèm layer gốc. Vẽ đen → ẩn vùng đó. Vẽ trắng → hiện lại. Không xóa pixel gốc — có thể hoàn tác bất kỳ lúc nào.

// Thêm Layer Mask Bấm tên layer → Mask → Thumbnail mask (màu trắng) xuất hiện bên cạnh thumbnail layer // Quy tắc Mask Vẽ đen lên mask → ẩn (transparent) Vẽ trắng lên mask → hiện (visible) Xám → bán trong suốt // Nhớ bấm vào thumbnail MASK trước khi vẽ!
⚠️ Dễ nhầm: phải bấm chọn thumbnail mask (có viền xanh) mới vẽ lên mask. Nếu không, bạn đang vẽ thẳng lên layer gốc.
Mask Clipping Mask — Tô màu trong hình dưới Bước 3 ▼

Clipping Mask "ghim" layer trên vào hình dạng của layer dưới. Mọi thứ vẽ trên layer clip chỉ hiện trong vùng có màu của layer bên dưới.

// Bật Clipping Mask Bấm tên layer → Clipping Mask → Mũi tên nhỏ xuất hiện, chỉ vào layer bên dưới // Use case kinh điển Layer 3: Highlight (Clipping Mask) ← chỉ hiện trong skin Layer 2: Shadow (Clipping Mask) ← chỉ hiện trong skin Layer 1: Skin base color
💡 Stack nhiều Clipping Mask lên 1 base layer → tô bóng sáng cực tự do, không bao giờ ra ngoài vùng màu gốc.
Adjustments Adjustments — Bộ lọc & hiệu chỉnh Bước 3 ▼

Bấm icon ✦ để mở Adjustments. Dùng để chỉnh màu, thêm blur, glow, noise cho layer.

// Hay dùng nhất Hue/Saturation/Brightness → Chỉnh tone màu tổng thể Color Balance → Làm ấm / lạnh màu Curves → Kiểm soát sáng/tối chuyên nghiệp Gaussian Blur → Làm mờ mượt (background bokeh) Motion Blur → Hiệu ứng tốc độ Glow → Ánh sáng phát sáng Noise → Thêm hạt film / texture // Dùng Apple Pencil để chỉnh cục bộ → Chọn Pencil mode → vẽ vào vùng muốn apply
🎨 Curves → kéo điểm giữa xuống → ảnh tối phim-noir. Kéo lên → ảnh overexposed dreamy.
Phần 7 · Text & Animation
Text Text Tool — Thêm chữ vào tranh Bước 2 ▼

Actions → Add → Add Text. Gõ chữ rồi dùng Style panel để chỉnh font, size, spacing.

// Text workflow 1. Actions → Add → Add Text 2. Gõ nội dung → bàn phím hiện lên 3. Bấm Aa → Style panel: - Font (import .ttf, .otf từ Files app) - Size, Tracking (letter-spacing), Leading (line-height) - Kern, Baseline 4. Rasterize khi xong → chuyển thành pixels để vẽ thêm lên
✅ Import font: mở file .ttf trong Files app → Chia sẻ → Procreate. Font sẽ có trong danh sách.
Animation Animation Assist — Vẽ hoạt hình đơn giản Bước 4 ▼

Procreate có tính năng vẽ frame-by-frame animation. Mỗi layer/group = 1 frame. Bật: Actions → Canvas → Animation Assist.

// Animation workflow cơ bản 1. Bật Animation Assist → thanh timeline xuất hiện dưới 2. Mỗi layer = 1 frame 3. Bật Onion Skinning → thấy frame trước mờ → dễ canh 4. Vẽ từng frame 5. Chỉnh FPS (4–12fps cho animation thủ công) 6. Export → Animated GIF hoặc Animated MP4 // Tips → Background layer: đánh dấu là Background → không tính là frame
💡 Loop animation + 6–8 frame đủ cho sticker telegram / avatar động đơn giản.
Phần 8 · Export & Advanced
Export Share & Export — Xuất file Bước 1 ▼

Actions → Share để xuất file. Có nhiều định dạng, chọn đúng mục đích.

// Định dạng & mục đích Procreate (.procreate) → Backup toàn bộ layer (chỉ mở trên Procreate) PSD → Mở trên Photoshop, giữ layer PDF → In ấn chất lượng cao PNG → Ảnh trong suốt, web, sticker JPEG → Ảnh nhỏ, đăng mạng xã hội TIFF → In ấn chuyên nghiệp // Nhanh: Share → PNG → Save to Photos
📌 Luôn giữ file .procreate gốc. Export PNG/JPG chỉ là bản phân phối — mất layer.
Export Time-lapse — Xuất video quá trình vẽ Bước 1 ▼

Procreate tự động ghi lại mọi nét vẽ từ đầu. Bạn có thể xuất ra video time-lapse bất kỳ lúc nào.

// Xuất time-lapse Actions → Video → Export Time-lapse → Full Length: video đầy đủ từng giây → 30 seconds: cắt gọn cho social media // Replay Actions → Video → Time-lapse Replay → Xem lại toàn bộ quá trình vẽ ngay trong app
✅ Đăng time-lapse lên TikTok / Instagram Reels → tăng follow nghệ sĩ cực hiệu quả.
Advanced Liquify — Biến dạng tự do như nước Bước 4 ▼

Adjustments → Liquify. Công cụ kéo dãn pixel như chất lỏng. Dùng để chỉnh hình dạng, tóc, quần áo bay.

// Liquify tools Push → Đẩy pixel theo hướng bút Pull → Kéo pixel lại Twirl → Xoáy tròn (Left/Right) Pinch → Kéo chụm vào giữa Expand → Phồng ra ngoài Crystals → Vỡ vụn nghệ thuật Reconstruct → Hoàn tác từng vùng // Chỉnh độ mạnh: Size + Distortion + Pressure
🎨 Liquify nhẹ nhàng trên khuôn mặt nhân vật → chỉnh tỉ lệ mắt, mũi, miệng mà không vẽ lại.
Advanced Snapping & Magnetic — Canh chỉnh chính xác Bước 4 ▼

Khi Transform, bật Snapping để object tự canh vào giữa, mép, hoặc các object khác chính xác không cần đo.

// Bật Snapping Vào Transform → bật Snapping (thanh dưới màn hình) → Snap to Canvas: canh mép canvas → Snap to Other Layers: canh theo layer khác → Distance: canh cách đều giữa các object // Đường guide xanh/vàng = đang snap
💡 Bật Snapping khi làm layout, poster, pattern → không cần căn thước tay, chính xác tuyệt đối.
Advanced Procreate Workflow — Quy trình vẽ nhân vật Bước 5 ▼

Workflow chuẩn từ sketch đến finish cho vẽ nhân vật anime/illustration.

// Quy trình 6 bước 1. Sketch thô → 6B Pencil, opacity 40%, layer riêng 2. Lineart → Studio Pen, layer mới trên sketch → Giảm opacity sketch xuống 20% để thấy rõ 3. Base color → Flat color, ColorDrop + Reference → Mỗi vùng màu 1 layer (da, tóc, áo, mắt...) 4. Shadow → Clipping Mask lên base, Multiply 50% 5. Highlight → Clipping Mask, Add hoặc Screen 6. Detail & FX → Rim light, glow, texture, background
✅ Đây là workflow tối ưu nhất: tách riêng từng công đoạn → dễ sửa, không đè nhau.
Advanced Procreate Pocket vs Procreate — Khác biệt Ghi chú ▼

Có 2 phiên bản Procreate cho thiết bị Apple khác nhau.

Procreate → iPad (full tính năng, khuyên dùng) Procreate Pocket → iPhone (nhỏ hơn, ít tính năng hơn) // Điểm khác nhau chính Procreate iPad: ✓ Nhiều layer hơn ✓ Hỗ trợ Apple Pencil đầy đủ ✓ Canvas lớn hơn ✓ Animation Assist đầy đủ Procreate Pocket: ✓ Dùng ngón tay / stylus thường ✗ Giới hạn hơn về canvas và layer
📌 Mua 1 lần, dùng mãi — không có subscription. Procreate iPad ~$12.99 USD.
Phần 9 · Pro Tips & Kỹ thuật nâng cao — Lên level thật sự
Pro Tips StreamLine — Làm mượt nét vẽ run tay Pro ▼

StreamLine là bộ ổn định nét vẽ built-in trong mỗi brush. Tăng StreamLine → nét vẽ mượt hơn, bù lại tay run nhẹ. Đây là bí quyết số 1 để có lineart đẹp.

// Bật StreamLine Brush Studio → Stroke → StreamLine → Kéo thanh từ 0% → 50–80% // Mức khuyên dùng theo mục đích 0–20% → Sketch thô, nét có hồn / không hoàn hảo 40–60% → Lineart nhân vật, nét chuẩn đẹp 80–100% → Calligraphy, lettering, nét như máy // Lưu ý: StreamLine cao → độ trễ nhiều hơn khi vẽ
🎯 Duplicate brush yêu thích → tạo bản "High Stabilization" riêng cho lineart, bản "Raw" riêng cho sketch.
Pro Tips Dynamic Brush Scaling — Vẽ chi tiết nhỏ không đổi size Pro ▼

Pro Tips Stabilization — 3 lớp ổn định nét vẽ Pro ▼
