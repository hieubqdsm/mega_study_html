Bài tập Thực hành Module 2 (Growth Marketing Pipeline)

Đề bài: Xây dựng hệ thống "AI Marketing Agency tự động" gồm 3 Agents chạy phối hợp:

  • Agent 1 (Researcher): Nhận từ khóa sản phẩm, lên mạng search Google tìm top 3 đối thủ.
  • Agent 2 (Copywriter): Nhận dữ liệu của Agent 1, viết 3 kịch bản quảng cáo đánh trúng yếu điểm đối thủ.
  • Agent 3 (Editor/Manager): Đóng vai Giám đốc Marketing, duyệt bài viết. Nếu bài viết không đạt tiêu chuẩn, tự động gửi trả lại Agent 2 kèm nhận xét để viết lại.

Bài 5: Kiến Trúc Đồ Thị Trạng Thái (State Graph)

Bài tập Thực hành Module 3 (Self-Healing AI Coder)

Đề bài: Tạo một hệ thống AI viết code có khả năng tự chạy thử và sửa lỗi:

  • Node 1 (Write Code): AI nhận yêu cầu bài toán toán học và viết mã Python.
  • Node 2 (Execute Code): Hệ thống tự động chạy file python đó trong môi trường ảo sandbox.
  • Rẽ nhánh (Conditional Router): Nếu chạy ra lỗi -> Chuyển hướng quay lại Node 1 kèm log lỗi để AI tự sửa (Cho phép lặp tối đa 3 lần). Nếu chạy thành công -> Xuất ra file code hoàn chỉnh.
