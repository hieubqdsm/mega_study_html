Chương 1 PyTorch là gì & cài đặt

Vì sao PyTorch thống trị giới nghiên cứu & ngày càng mạnh ở production.

PyTorch (Meta AI) là thư viện học sâu dựa trên tensor với autograd động (define-by-run): đồ thị tính toán dựng khi chạy, nên debug bằng print/breakpoint như Python thường. Đó là lý do nó được giới nghiên cứu & phần lớn dự án LLM ưa chuộng.

PyTorch vs TensorFlow Khác biệt lớn nhất: ở PyTorch bạn viết vòng lặp huấn luyện (tường minh); ở Keras model.fit làm hộ (ngắn gọn). Hiểu PyTorch giúp bạn nắm rõ bên trong mọi framework.

Chương 2 Tensor: shape, dtype, device

Giống NumPy nhưng chạy được trên GPU & có autograd.
python●
import torch

a = torch.tensor([[1., 2.], [3., 4.]])
print(a.shape, a.dtype)         # torch.Size([2, 2]) torch.float32

b = torch.tensor([[10.], [20.]])
print(a @ b)                    # nhân ma trận
print(a.mean(), a.sum(dim=0))   # trung bình; tổng theo cột
print(a.view(4))                # reshape → vector 4 (như tf.reshape)

# Chuyển sang GPU nếu có
device = 'cuda' if torch.cuda.is_available() else 'cpu'
a = a.to(device)                # mọi tensor cùng phép toán phải cùng device

# Qua lại với NumPy
import numpy as np
n = a.cpu().numpy()             # tensor → numpy
t = torch.from_numpy(np.ones(3))
Lỗi kinh điển RuntimeError: Expected all tensors to be on the same device — bạn quên .to(device) cho dữ liệu hoặc model. Quy tắc: model và batch phải ở cùng device.
✍️ Bài tập 1 — Tạo & biến đổi tensor
Tạo tensor x shape (3, 4) toàn số 1 bằng torch.ones, rồi tính tổng theo trục cột bằng x.sum(dim=0).
Xem lời giải
x = torch.ones(3, 4)
col_sum = x.sum(dim=0)   # → tensor([3., 3., 3., 3.])
print(col_sum)

Chương 3 Autograd: backward()

Tự động tính đạo hàm — nền tảng của huấn luyện.

Đặt requires_grad=True cho tensor cần đạo hàm. Sau khi tính ra một scalar (loss), gọi .backward() để PyTorch lan truyền ngược & điền .grad.

python●
w = torch.tensor(3.0, requires_grad=True)
y = w ** 2                 # y = w²
y.backward()               # tính dy/dw
print(w.grad)              # 2w = 6.0

# Một bước cập nhật thủ công (không theo dõi gradient)
with torch.no_grad():
    w -= 0.1 * w.grad
w.grad.zero_()             # XÓA grad trước bước sau (grad bị cộng dồn!)
Gradient bị cộng dồn PyTorch cộng dồn .grad qua mỗi backward(). Vì vậy mỗi vòng lặp phải gọi optimizer.zero_grad() (hoặc .grad.zero_()) — quên bước này là lỗi thầm lặng phổ biến nhất.
✍️ Bài tập 2 — Tính gradient
Cho x = torch.tensor(2.0, requires_grad=True). Tính y = x**3 + 2*x, gọi y.backward() rồi in x.grad (đúng = 14).
Xem lời giải
y = x**3 + 2*x
y.backward()
print(x.grad)   # 3x² + 2 = 14.0

Chương 4 nn.Module — model đầu tiên

Mọi model PyTorch là một lớp kế thừa nn.Module.

Định nghĩa lớp con của nn.Module: khai báo các lớp trong __init__, mô tả luồng dữ liệu trong forward.

python · MLP cho MNIST●
import torch.nn as nn
import torch.nn.functional as F

class Net(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 128)
        self.drop = nn.Dropout(0.2)
        self.fc2 = nn.Linear(128, 10)
    def forward(self, x):
        x = x.view(x.size(0), -1)    # phẳng ảnh → (batch, 784)
        x = F.relu(self.fc1(x))
        x = self.drop(x)
        return self.fc2(x)              # trả logits (chưa softmax)

model = Net()
Quy ước forward thường trả về logits thô. Hàm nn.CrossEntropyLoss đã gộp sẵn softmax + log + NLL, nên không thêm softmax ở cuối khi phân loại.
✍️ Bài tập 3 — Định nghĩa nn.Module
Viết lớp MLP(nn.Module): trong __init__ tạo self.fc1 = nn.Linear(64, 32) và self.fc2 = nn.Linear(32, 1); trong forward trả về self.fc2(F.relu(self.fc1(x))). Nhớ gọi super().__init__().
Xem lời giải
class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(64, 32)
        self.fc2 = nn.Linear(32, 1)
    def forward(self, x):
        return self.fc2(F.relu(self.fc1(x)))

Chương 5 Vòng lặp huấn luyện chuẩn

5 dòng "thần chú" bạn sẽ viết cả nghìn lần.
python · training loop●
model = Net().to(device)
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

for epoch in range(EPOCHS):
    model.train()                        # bật dropout/batchnorm
    for X, y in train_loader:
        X, y = X.to(device), y.to(device)
        optimizer.zero_grad()            # 1. xóa grad cũ
        out = model(X)                   # 2. forward
        loss = criterion(out, y)         # 3. tính loss
        loss.backward()                  # 4. backward (tính grad)
        optimizer.step()                 # 5. cập nhật trọng số
Học thuộc 5 bước zero_grad → forward → loss → backward → step. Đây là khác biệt cốt lõi với Keras: bạn thấy & kiểm soát từng bước.
✍️ Bài tập 4 — Hoàn thiện 5 bước
Điền thân vòng lặp: optimizer.zero_grad(), out = model(X), loss = criterion(out, y), loss.backward(), optimizer.step().
Xem lời giải
    optimizer.zero_grad()
    out = model(X)
    loss = criterion(out, y)
    loss.backward()
    optimizer.step()

Chương 6 Dataset & DataLoader

Nạp dữ liệu theo lô, xáo trộn, song song.
python●
from torch.utils.data import DataLoader
from torchvision import datasets, transforms

tf = transforms.Compose([
    transforms.ToTensor(),                       # [0,255] → [0,1], HWC→CHW
    transforms.Normalize((0.5,), (0.5,)),
])
train_ds = datasets.MNIST('.', train=True, download=True, transform=tf)
train_loader = DataLoader(train_ds, batch_size=64, shuffle=True, num_workers=2)

for X, y in train_loader:
    print(X.shape)    # torch.Size([64, 1, 28, 28])
    break
CHW không phải HWC PyTorch để channel trước: (batch, channels, height, width). Khác với TensorFlow (channels-last). ToTensor() tự lo việc hoán vị này.
✍️ Bài tập 5 — Tạo DataLoader
Tạo train_loader = DataLoader(train_ds, ...) với batch_size=64 và shuffle=True.
Xem lời giải

Chương 8 CNN cho ảnh

nn.Conv2d, pooling, và quy luật giảm kích thước.
python · CIFAR-10●
class CNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(3, 32, 3, padding=1)
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
        self.pool  = nn.MaxPool2d(2)
        self.fc    = nn.Linear(64 * 8 * 8, 10)
    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x)))   # 32x32 → 16x16
        x = self.pool(F.relu(self.conv2(x)))   # 16x16 → 8x8
        x = torch.flatten(x, 1)
        return self.fc(x)
Tính kích thước Flatten Phải tự tính số chiều vào nn.Linear: sau 2 lần pool, 32×32 → 8×8, với 64 kênh ⇒ 64*8*8. Sai số này gây lỗi shape khi forward — mẹo: dùng nn.LazyLinear để PyTorch tự suy.
✍️ Bài tập 7 — Lớp tích chập
Khai báo self.conv = nn.Conv2d(3, 16, 3, padding=1) và self.pool = nn.MaxPool2d(2).
Xem lời giải
self.conv = nn.Conv2d(3, 16, 3, padding=1)
self.pool = nn.MaxPool2d(2)

Chương 9 Transfer learning

Dùng ResNet đã học ImageNet cho bài toán của bạn.
python●
from torchvision import models

model = models.resnet18(weights='IMAGENET1K_V1')
for p in model.parameters():
    p.requires_grad = False           # đóng băng backbone

model.fc = nn.Linear(model.fc.in_features, 5)   # thay đầu phân loại 5 lớp
# Chỉ tham số của model.fc sẽ được học
optimizer = torch.optim.Adam(model.fc.parameters(), lr=1e-3)
Mẫu fine-tune (1) Đóng băng + train đầu mới. (2) Mở băng vài block cuối với lr nhỏ. Thay model.fc tự động bật requires_grad=True cho lớp mới.

PyTorch không có EarlyStopping sẵn — ta tự cài: theo dõi val_loss, lưu trọng số tốt nhất, dừng khi không cải thiện sau patience epoch.

python · early stopping thủ công●
best, wait = float('inf'), 0
for epoch in range(100):
    val = validate(model, val_loader)
    if val < best:
        best, wait = val, 0
        torch.save(model.state_dict(), 'best.pth')
    else:
        wait += 1
        if wait >= 3: break      # dừng sớm

Chương 12 Custom Dataset & loss

Khi dữ liệu của bạn không có sẵn loader.
python · Dataset tùy biến●
from torch.utils.data import Dataset

class MyDataset(Dataset):
    def __init__(self, X, y):
        self.X, self.y = X, y
    def __len__(self):
        return len(self.X)
    def __getitem__(self, i):
        return self.X[i], self.y[i]   # trả 1 mẫu
3 method bắt buộc __init__, __len__, __getitem__. DataLoader sẽ tự gọi __getitem__ & gộp batch. Loss tùy biến chỉ là một hàm nhận (pred, target) trả về scalar có thể backward().

Chương 13 RNN/LSTM & NLP

Xử lý chuỗi: văn bản, thời gian.
python · phân loại cảm xúc●
class TextClassifier(nn.Module):
    def __init__(self, vocab, emb=128, hid=64):
        super().__init__()
        self.emb = nn.Embedding(vocab, emb)
        self.lstm = nn.LSTM(emb, hid, batch_first=True, bidirectional=True)
        self.fc = nn.Linear(hid * 2, 1)
    def forward(self, x):
        x = self.emb(x)
        _, (h, _) = self.lstm(x)          # lấy hidden state cuối
        h = torch.cat([h[-2], h[-1]], dim=1)  # ghép 2 chiều
        return self.fc(h)
batch_first=True Cho input shape (batch, seq, feature) thay vì mặc định (seq, batch, feature) — dễ đọc hơn nhiều. Dùng nn.utils.rnn.pad_sequence cho câu dài ngắn khác nhau.

Chương 14 Mixed precision & đa GPU

Nhanh hơn, tiết kiệm VRAM, mở rộng quy mô.
python · AMP (automatic mixed precision)●
scaler = torch.cuda.amp.GradScaler()
for X, y in loader:
    optimizer.zero_grad()
    with torch.autocast(device_type='cuda'):   # tự dùng float16 chỗ an toàn
        loss = criterion(model(X), y)
    scaler.scale(loss).backward()
    scaler.step(optimizer)
    scaler.update()

Chương 16 Dự án thực chiến tổng hợp

Phân loại ảnh 5 lớp end-to-end bằng transfer learning.
  1. Dữ liệu: datasets.ImageFolder + transform (resize, augment, normalize) → DataLoader train/val.
  2. Model: resnet18(weights=...), đóng băng backbone, thay model.fc = Linear(..., 5).
  3. Loss/Opt: CrossEntropyLoss + Adam(model.fc.parameters()).
  4. Train: vòng lặp 5 bước + early stopping thủ công, chuyển .to(device).
  5. Đánh giá: model.eval() + torch.no_grad(), tính accuracy.
  6. Triển khai: xuất ONNX hoặc TorchScript.
✍️ Bài tập 9 — Vòng đánh giá đúng chuẩn
Viết phần đánh giá: gọi model.eval(), bọc with torch.no_grad(): rồi forward out = model(X).
Xem lời giải
model.eval()
with torch.no_grad():
    for X, y in val_loader:
        out = model(X)
Hoàn thành khóa học 🎉 Bạn đã đi từ tensor → triển khai. Bước tiếp: làm lại với dữ liệu của bạn, thử torch.compile, và đối chiếu với khóa TensorFlow & Keras để thấy hai trường phái giải cùng bài toán.
