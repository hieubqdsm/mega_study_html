Chương 2 pipeline() — suy luận 1 dòng

Cách nhanh nhất để chạy một model AI.
python●
from transformers import pipeline

clf = pipeline("sentiment-analysis")        # tự tải model mặc định
clf("Khóa học này tuyệt vời!")
# [{'label': 'POSITIVE', 'score': 0.99}]

gen = pipeline("text-generation", model="gpt2")
qa  = pipeline("question-answering")
asr = pipeline("automatic-speech-recognition")   # âm thanh → chữ
pipeline = tokenizer + model + hậu xử lý Một lệnh lo trọn: tiền xử lý input, chạy model, giải mã output. Có cho mọi tác vụ: phân loại, NER, tóm tắt, dịch, sinh ảnh-văn bản, ASR... Chỉ định model="..." để chọn model cụ thể từ Hub.
✍️ Bài tập 1 — Tạo pipeline
Tạo pipeline("text-classification", model="distilbert-base-uncased-finetuned-sst-2-english") rồi gọi nó với một câu.
Xem lời giải
clf = pipeline("text-classification",
               model="distilbert-base-uncased-finetuned-sst-2-english")

Chương 4 Tokenizer

Văn bản → số (token id) mà model hiểu.
python●
enc = tokenizer("Hello world", return_tensors="pt")
print(enc["input_ids"])        # tensor các token id
print(tokenizer.decode(enc["input_ids"][0]))   # giải mã ngược

# Theo lô: padding để cùng độ dài, truncation để cắt câu dài
batch = tokenizer(["câu 1", "câu dài hơn..."],
                  padding=True, truncation=True, return_tensors="pt")
Subword tokenization Tokenizer cắt từ thành "mảnh" (subword) để xử lý từ lạ. Trả về input_ids (số) + attention_mask (1=token thật, 0=padding). return_tensors="pt" cho PyTorch tensor.
✍️ Bài tập 3 — Token hóa theo lô
Gọi tokenizer(texts, padding=True, truncation=True, return_tensors="pt").
Xem lời giải
enc = tokenizer(texts, padding=True, truncation=True, return_tensors="pt")

Chương 5 Model & AutoModel

Chọn đúng "đầu" model cho tác vụ.
LớpDùng cho
AutoModelLấy embedding/hidden states thô
AutoModelForSequenceClassificationPhân loại câu
AutoModelForTokenClassificationNER, gán nhãn token
AutoModelForCausalLMSinh văn bản (GPT, Llama)
AutoModelForSeq2SeqLMDịch, tóm tắt (T5, BART)
python●
from transformers import AutoModelForSequenceClassification
import torch

model = AutoModelForSequenceClassification.from_pretrained(name, num_labels=2)
with torch.no_grad():
    logits = model(**enc).logits      # điểm thô từng lớp
probs = logits.softmax(dim=-1)        # → xác suất
logits ≠ xác suất Model trả logits (số thực, chưa chuẩn hóa). Áp softmax để ra xác suất, argmax để ra nhãn dự đoán.

Chương 6 Thư viện Datasets

Nạp & xử lý dữ liệu ở quy mô lớn.
python●
from datasets import load_dataset

ds = load_dataset("imdb")            # tải từ Hub → DatasetDict(train/test)
ds["train"][0]                       # {'text': ..., 'label': 1}

def tok(batch): return tokenizer(batch["text"], truncation=True)
ds_tok = ds.map(tok, batched=True)    # áp tokenizer cho cả dataset
map(batched=True) Xử lý theo lô, song song, memory-mapped (không nạp hết vào RAM) → cực nhanh với dữ liệu lớn. Dùng streaming=True để duyệt dataset khổng lồ mà không tải hết.
✍️ Bài tập 4 — Nạp & token hóa dataset
Tải load_dataset("imdb") rồi áp tokenizer bằng ds.map(tok, batched=True).
Xem lời giải
ds = load_dataset("imdb")
ds_tok = ds.map(tok, batched=True)

Chương 7 Suy luận đúng cách

Nhanh, đúng device, không rò bộ nhớ.
python●
device = "cuda" if torch.cuda.is_available() else "cpu"
model.to(device).eval()                       # chế độ suy luận
enc = {k: v.to(device) for k, v in enc.items()}
with torch.no_grad():
    out = model(**enc)
pred = out.logits.argmax(-1)                  # nhãn dự đoán
3 thứ luôn nhớ khi suy luận .eval() (tắt dropout), torch.no_grad() (không tính grad → nhanh, ít RAM), và đưa cả model lẫn input về cùng device.

Chương 8 Trainer & TrainingArguments

Vòng lặp huấn luyện "pin sẵn" của HF.
python●
from transformers import TrainingArguments, Trainer

args = TrainingArguments(
    output_dir="out", learning_rate=2e-5,
    per_device_train_batch_size=16, num_train_epochs=3,
    eval_strategy="epoch", save_strategy="epoch",
)
trainer = Trainer(model=model, args=args,
    train_dataset=ds_tok["train"], eval_dataset=ds_tok["test"],
    tokenizer=tokenizer)
trainer.train()
Trainer lo hết Vòng lặp, backward, optimizer, scheduler, logging, lưu checkpoint, đánh giá, đa GPU — không cần tự viết. Tùy biến qua TrainingArguments (hàng trăm tham số) hoặc override method của Trainer.
✍️ Bài tập 5 — Cấu hình huấn luyện
Tạo TrainingArguments(output_dir="out", num_train_epochs=3, per_device_train_batch_size=16).
Xem lời giải
args = TrainingArguments(output_dir="out", num_train_epochs=3,
                         per_device_train_batch_size=16)

Chương 9 Data collator & metrics

Gộp lô động & đo chất lượng.
python●
from transformers import DataCollatorWithPadding
import evaluate, numpy as np

collator = DataCollatorWithPadding(tokenizer)   # padding theo lô (hiệu quả hơn)
acc = evaluate.load("accuracy")
def compute_metrics(p):
    preds = np.argmax(p.predictions, axis=-1)
    return acc.compute(predictions=preds, references=p.label_ids)
Dynamic padding DataCollatorWithPadding chỉ pad đến câu dài nhất trong từng lô (thay vì pad toàn bộ đến max) → nhanh hơn nhiều. Truyền data_collator & compute_metrics vào Trainer.

Chương 10 Fine-tune phân loại (end-to-end)

Ghép mọi mảnh thành một quy trình hoàn chỉnh.
  1. Dữ liệu: load_dataset → map(tok, batched=True).
  2. Model: AutoModelForSequenceClassification(..., num_labels=N).
  3. Cấu hình: TrainingArguments + DataCollatorWithPadding + compute_metrics.
  4. Huấn luyện: Trainer(...).train().
  5. Đánh giá: trainer.evaluate().
Mẹo Bắt đầu với model nhỏ (distilbert) & tập con dữ liệu để chạy thử nhanh, rồi mới mở rộng. Đặt fp16=True (hoặc bf16=True) trong TrainingArguments để train nhanh hơn trên GPU.
✍️ Bài tập 6 — Ráp Trainer
Tạo Trainer(model=model, args=args, train_dataset=ds_tok["train"], eval_dataset=ds_tok["test"]) rồi .train().
Xem lời giải
trainer = Trainer(model=model, args=args,
    train_dataset=ds_tok["train"], eval_dataset=ds_tok["test"])
trainer.train()

Chương 12 Sinh văn bản (generate)

Điều khiển cách LLM tạo chữ.
python●
from transformers import AutoModelForCausalLM
lm = AutoModelForCausalLM.from_pretrained("gpt2")
ids = tokenizer("Once upon a time", return_tensors="pt").input_ids
out = lm.generate(ids, max_new_tokens=50, do_sample=True,
                  temperature=0.8, top_p=0.95)
print(tokenizer.decode(out[0], skip_special_tokens=True))
Tham sốTác dụng
max_new_tokensĐộ dài sinh thêm
do_sample=TrueLấy mẫu ngẫu nhiên (sáng tạo) vs greedy
temperatureCao = ngẫu nhiên/sáng tạo hơn
top_p / top_kGiới hạn tập token được chọn

Chương 13 Accelerate

Đa GPU & mixed precision không cần viết lại code.
python●
from accelerate import Accelerator
accelerator = Accelerator()
model, optimizer, dataloader = accelerator.prepare(model, optimizer, dataloader)
for batch in dataloader:
    loss = model(**batch).loss
    accelerator.backward(loss)     # thay loss.backward()
    optimizer.step(); optimizer.zero_grad()
# Chạy: accelerate launch train.py (tự lo đa GPU/TPU/mixed precision)
Accelerate Viết vòng lặp PyTorch một lần, chạy được trên CPU, 1 GPU, đa GPU, TPU mà gần như không đổi code. Trainer dùng Accelerate bên dưới.

Chương 14 Lượng tử hóa (bitsandbytes)

Nhét model lớn vào GPU nhỏ.
python · tải model 4-bit●
from transformers import AutoModelForCausalLM, BitsAndBytesConfig
import torch

bnb = BitsAndBytesConfig(load_in_4bit=True,
        bnb_4bit_quant_type="nf4", bnb_4bit_compute_dtype=torch.bfloat16)
model = AutoModelForCausalLM.from_pretrained(
        "mistralai/Mistral-7B-v0.1", quantization_config=bnb, device_map="auto")
4-bit = chạy 7B trên ~6GB VRAM Lượng tử hóa nén trọng số (float16 → 4-bit) giúp model khổng lồ vừa GPU phổ thông, đổi lại sai số nhỏ. Đây chính là nền của QLoRA — fine-tune model lớn trên 1 GPU (xem khóa PEFT & LoRA).

Chương 16 Dự án thực chiến
