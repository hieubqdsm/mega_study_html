Chương 1 TensorFlow là gì & cài đặt

Bức tranh tổng thể, hệ sinh thái và cách dựng môi trường.

TensorFlow là thư viện mã nguồn mở của Google cho học máy & học sâu. Nó cho phép định nghĩa các phép tính trên tensor (mảng nhiều chiều), tự động tính đạo hàm (autograd), chạy trên CPU/GPU/TPU, và triển khai mô hình ra nhiều nền tảng.

Keras là API cấp cao nằm trong TensorFlow (tf.keras) — cách chính thức & dễ nhất để xây mô hình. Trong khóa này ta dùng Keras làm mặc định và "hạ xuống" tầng thấp khi cần.

Chương 2 Tensor: shape, dtype, phép toán

Đơn vị dữ liệu nền tảng của mọi thứ trong TensorFlow.

Một tensor là mảng nhiều chiều có kiểu (dtype) đồng nhất. Vô hướng là tensor 0 chiều, vector là 1 chiều, ma trận 2 chiều, ảnh RGB là 3 chiều, một batch ảnh là 4 chiều (batch, height, width, channels).

python●
import tensorflow as tf

a = tf.constant([[1, 2], [3, 4]], dtype=tf.float32)
print(a.shape, a.dtype)        # (2, 2) float32

b = tf.constant([[10.0], [20.0]])
print(a @ b)                   # nhân ma trận → shape (2,1)
print(tf.reduce_mean(a))       # trung bình tất cả phần tử
print(tf.reshape(a, [4]))      # đổi shape → vector 4 phần tử

# Variable: tensor có thể thay đổi (dùng cho trọng số)
w = tf.Variable([[0.5, 0.5]])
w.assign_add([[0.1, 0.1]])     # cập nhật tại chỗ
Broadcasting Khi shape không khớp, TF tự "nong" chiều có kích thước 1 cho khớp: tf.constant([1,2,3]) + 10 → [11,12,13]. Hiểu broadcasting giúp tránh 80% lỗi shape.
✍️ Bài tập 1 — Tạo & biến đổi tensor
Viết code: tạo tensor x kiểu float32 shape (3, 4) toàn số 1 bằng tf.ones, rồi tính tổng theo trục cột (axis=0) bằng tf.reduce_sum.
Xem lời giải
x = tf.ones([3, 4], dtype=tf.float32)
col_sum = tf.reduce_sum(x, axis=0)   # → [3., 3., 3., 3.]
print(col_sum)

Chương 3 Autograd với GradientTape

Cơ chế tự động tính đạo hàm — trái tim của việc huấn luyện.

Huấn luyện = giảm hàm mất mát bằng gradient descent. TensorFlow ghi lại các phép tính trong một tf.GradientTape rồi tính đạo hàm của output theo các biến.

python●
w = tf.Variable(3.0)

with tf.GradientTape() as tape:
    y = w ** 2            # y = w²

dy_dw = tape.gradient(y, w)   # dy/dw = 2w = 6.0
print(dy_dw)                  # tf.Tensor(6.0)

# Một bước "học" thủ công: kéo w về phía giảm y
lr = 0.1
w.assign_sub(lr * dy_dw)      # w = 3 - 0.1*6 = 2.4
Vì sao quan trọng Mọi optimizer của Keras (SGD, Adam...) đều dùng cơ chế này bên dưới. Khi bạn cần vòng lặp huấn luyện tùy biến (chương 12), bạn sẽ tự viết GradientTape.
✍️ Bài tập 2 — Tính gradient
Cho x = tf.Variable(2.0). Dùng tf.GradientTape tính đạo hàm của y = x**3 + 2*x theo x (kết quả đúng = 14). Gọi tape.gradient(...).
Xem lời giải
with tf.GradientTape() as tape:
    y = x**3 + 2*x
grad = tape.gradient(y, x)   # 3x² + 2 = 14.0
print(grad)

Chương 4 Mạng nơ-ron đầu tiên (Sequential)

Phân loại chữ số viết tay MNIST trong ~15 dòng.

keras.Sequential xếp chồng các lớp tuyến tính. Đây là cách nhanh nhất cho mạng "một đầu vào, một đầu ra".

python · MNIST●
import tensorflow as tf
from tensorflow import keras

(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()
x_train, x_test = x_train / 255.0, x_test / 255.0   # chuẩn hóa về [0,1]

model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),     # ảnh 28x28 → vector 784
    keras.layers.Dense(128, activation='relu'),   # lớp ẩn
    keras.layers.Dropout(0.2),
    keras.layers.Dense(10, activation='softmax')  # 10 lớp đầu ra
])
model.summary()
Các activation hay gặp relu cho lớp ẩn; softmax cho phân loại nhiều lớp (đầu ra là xác suất); sigmoid cho phân loại nhị phân; không activation (linear) cho hồi quy.
✍️ Bài tập 3 — Dựng Sequential
Tạo một keras.Sequential cho bài hồi quy: đầu vào 8 đặc trưng → Dense(64, relu) → Dense(64, relu) → Dense(1) (không activation). Phải dùng keras.Sequential và layers.Dense.
Xem lời giải
model = keras.Sequential([
    layers.Input(shape=(8,)),
    layers.Dense(64, activation='relu'),
    layers.Dense(64, activation='relu'),
    layers.Dense(1)
])

Chương 5 compile / fit / evaluate / predict

Vòng đời huấn luyện chuẩn của Keras.

Sau khi dựng model, ta cần 3 thứ trong compile: optimizer (cách cập nhật trọng số), loss (đo sai số), metrics (theo dõi).

python●
model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',   # nhãn là số nguyên 0..9
    metrics=['accuracy']
)

history = model.fit(x_train, y_train,
                    epochs=5, batch_size=32,
                    validation_split=0.1)        # giữ 10% để kiểm định

test_loss, test_acc = model.evaluate(x_test, y_test)
preds = model.predict(x_test[:3])                # xác suất 10 lớp
Bài toánLoss thường dùngActivation cuối
Phân loại nhiều lớp (nhãn số)sparse_categorical_crossentropysoftmax
Phân loại nhiều lớp (one-hot)categorical_crossentropysoftmax
Phân loại nhị phânbinary_crossentropysigmoid
Hồi quymse / maelinear
Bẫy thường gặp Chọn nhầm cặp loss/activation → model "học" nhưng accuracy đứng yên. Ví dụ dùng categorical_crossentropy với nhãn dạng số nguyên (phải dùng bản sparse_).
✍️ Bài tập 4 — Compile cho phân loại nhị phân
Gọi model.compile(...) cho bài phân loại nhị phân: optimizer 'adam', loss 'binary_crossentropy', theo dõi 'accuracy'.
Xem lời giải
model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)

Chương 6 Dữ liệu với tf.data

Pipeline dữ liệu nhanh, không nghẽn GPU.

tf.data.Dataset giúp nạp, xáo trộn, gộp batch và nạp trước dữ liệu hiệu quả — đặc biệt khi dữ liệu lớn hơn RAM.

python●
ds = tf.data.Dataset.from_tensor_slices((x_train, y_train))
ds = (ds
      .shuffle(10_000)          # xáo trộn
      .batch(32)                # gộp lô 32 mẫu
      .prefetch(tf.data.AUTOTUNE))   # nạp trước lô kế trong khi GPU bận

model.fit(ds, epochs=5)         # truyền thẳng Dataset vào fit
Thứ tự vàng map → cache → shuffle → batch → prefetch. Đặt cache() sau bước tiền xử lý tốn kém để chỉ tính một lần.
✍️ Bài tập 5 — Dựng pipeline tf.data
Từ ds = tf.data.Dataset.from_tensor_slices((X, y)), hãy nối chuỗi: shuffle(1000), batch(64), prefetch(tf.data.AUTOTUNE).
Xem lời giải
ds = ds.shuffle(1000).batch(64).prefetch(tf.data.AUTOTUNE)

Chương 7 Overfitting & callbacks

Khi model học thuộc lòng thay vì khái quát hóa.

Overfitting: loss train giảm nhưng loss validation tăng. Vũ khí: nhiều dữ liệu hơn, Dropout, regularization L1/L2, và dừng sớm.

python · callbacks●
from tensorflow.keras import callbacks

cbs = [
    callbacks.EarlyStopping(monitor='val_loss', patience=3,
                            restore_best_weights=True),
    callbacks.ModelCheckpoint('best.keras', save_best_only=True),
    callbacks.ReduceLROnPlateau(monitor='val_loss', factor=0.5, patience=2),
]
model.fit(x_train, y_train, epochs=100,
          validation_split=0.2, callbacks=cbs)
EarlyStopping Đặt epochs lớn rồi để EarlyStopping tự dừng khi val_loss ngừng cải thiện sau patience epoch — vừa nhanh vừa tránh overfit.

2. Callback nào tự dừng huấn luyện khi không cải thiện?
✅ EarlyStopping theo dõi một metric và dừng khi hết kiên nhẫn (patience).

Chương 8 Functional API

Khi Sequential không đủ: nhiều input/output, nhánh, chia sẻ lớp.

inputs = keras.Input(shape=(784,))
x = keras.layers.Dense(128, activation='relu')(inputs)
x = keras.layers.Dense(64, activation='relu')(x)

inputs = keras.Input(shape=(32,))

Chương 9 CNN cho ảnh

Convolutional Neural Network — chuẩn vàng cho thị giác máy tính.

Lớp Conv2D quét bộ lọc trên ảnh để học đặc trưng cục bộ (cạnh, góc, texture); MaxPooling2D giảm kích thước & tăng bất biến dịch chuyển.

python · CIFAR-10●
model = keras.Sequential([
    keras.layers.Input(shape=(32, 32, 3)),
    keras.layers.Conv2D(32, 3, activation='relu'),
    keras.layers.MaxPooling2D(),
    keras.layers.Conv2D(64, 3, activation='relu'),
    keras.layers.MaxPooling2D(),
    keras.layers.Flatten(),
    keras.layers.Dense(64, activation='relu'),
    keras.layers.Dense(10, activation='softmax'),
])
Quy luật kiến trúc Càng sâu, số filter càng tăng (32 → 64 → 128) trong khi không gian (H×W) giảm dần qua pooling. Kết thúc bằng Flatten + Dense để ra quyết định.
✍️ Bài tập 7 — Khối CNN
Thêm vào danh sách lớp một khối: Conv2D(64, 3, relu) theo sau là MaxPooling2D(). Dùng đúng tên lớp Conv2D và MaxPooling2D.
Xem lời giải
    keras.layers.Conv2D(64, 3, activation='relu'),

Chương 10 Transfer learning & augmentation

Đứng trên vai người khổng lồ: dùng lại model đã huấn luyện.

Thay vì huấn luyện từ đầu (cần triệu ảnh), ta lấy một model mạnh đã học trên ImageNet (MobileNetV2, EfficientNet), đóng băng phần backbone và chỉ huấn luyện "đầu" phân loại mới.

python●
base = keras.applications.MobileNetV2(
    input_shape=(160, 160, 3), include_top=False, weights='imagenet')
base.trainable = False          # đóng băng backbone

model = keras.Sequential([
    keras.layers.RandomFlip('horizontal'),   # augmentation
    keras.layers.RandomRotation(0.1),
    base,
    keras.layers.GlobalAveragePooling2D(),
    keras.layers.Dropout(0.2),
    keras.layers.Dense(5, activation='softmax'),   # 5 lớp mới
])
Fine-tuning 2 pha (1) Huấn luyện đầu mới với backbone đóng băng. (2) Mở băng vài lớp cuối của backbone với learning_rate rất nhỏ để tinh chỉnh. Đảo thứ tự sẽ phá hỏng trọng số đã học.

Chương 12 Custom loop & subclass

Khi bạn cần toàn quyền kiểm soát.

Với GAN, RL, hay loss phức tạp, model.fit không đủ. Ta tự viết vòng lặp bằng GradientTape.

python · custom training loop●
opt = keras.optimizers.Adam()
loss_fn = keras.losses.SparseCategoricalCrossentropy()

for epoch in range(EPOCHS):
    for x_batch, y_batch in train_ds:
        with tf.GradientTape() as tape:
            logits = model(x_batch, training=True)
            loss = loss_fn(y_batch, logits)
        grads = tape.gradient(loss, model.trainable_variables)
        opt.apply_gradients(zip(grads, model.trainable_variables))
python · subclass Model●
class MyModel(keras.Model):
    def __init__(self):
        super().__init__()
        self.d1 = keras.layers.Dense(64, activation='relu')
        self.out = keras.layers.Dense(10)
    def call(self, x):
        return self.out(self.d1(x))
✍️ Bài tập 8 — Một bước cập nhật thủ công
Trong vòng lặp: tính grads = tape.gradient(loss, model.trainable_variables) rồi opt.apply_gradients(zip(grads, model.trainable_variables)).
Xem lời giải
grads = tape.gradient(loss, model.trainable_variables)
opt.apply_gradients(zip(grads, model.trainable_variables))

Chương 13 RNN/LSTM & văn bản

Dữ liệu tuần tự: câu chữ, chuỗi thời gian.
python · phân loại cảm xúc●
vectorizer = keras.layers.TextVectorization(max_tokens=20000, output_sequence_length=200)
vectorizer.adapt(train_texts)          # học từ vựng từ dữ liệu

model = keras.Sequential([
    vectorizer,
    keras.layers.Embedding(20000, 128),    # từ → vector
    keras.layers.Bidirectional(keras.layers.LSTM(64)),
    keras.layers.Dense(1, activation='sigmoid'),

Chương 14 tf.function & hiệu năng

Tăng tốc và tiết kiệm bộ nhớ.
python●
@tf.function            # biên dịch Python → graph, chạy nhanh hơn nhiều
def train_step(x, y):
    with tf.GradientTape() as tape:
        loss = loss_fn(y, model(x, training=True))
    grads = tape.gradient(loss, model.trainable_variables)
    opt.apply_gradients(zip(grads, model.trainable_variables))
    return loss

# Mixed precision: nhanh hơn ~2-3x trên GPU mới
keras.mixed_precision.set_global_policy('mixed_float16')

Chương 16 Dự án thực chiến tổng hợp

Ghép mọi mảnh: phân loại ảnh hoa 5 lớp end-to-end.
  1. Dữ liệu: keras.utils.image_dataset_from_directory → train/val Dataset, batch + prefetch.
  2. Model: augmentation + MobileNetV2 đóng băng + GlobalAveragePooling2D + Dense(5, softmax).
  3. Compile: Adam, sparse_categorical_crossentropy, accuracy.
  4. Train: EarlyStopping + ModelCheckpoint; rồi fine-tune vài lớp cuối.
  5. Đánh giá: confusion matrix, evaluate trên test.
  6. Triển khai: xuất .tflite để chạy trên điện thoại.
✍️ Bài tập 9 — Mảnh ghép cuối: train với callbacks
Gọi model.fit với train_ds, validation_data=val_ds, epochs=20 và truyền danh sách callbacks=[early_stop].
Xem lời giải
history = model.fit(
    train_ds,
    validation_data=val_ds,
    epochs=20,
    callbacks=[early_stop]
)
